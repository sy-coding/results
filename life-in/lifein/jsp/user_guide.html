#등록방법

STEP1 라이프인 오토패스워드 등록 가능한 사이트 확인
간편로그인 사이트 추가 선택
등록 가능한 사이트 확인
- "라이프인" 앱의 "간편로그인 사이트 추가"에서 "오토패스워드 간편로그인" 등록이 가능한 사이트를 확인합니다.
STEP2 사이트에서 라이프인 오토패스워드 설정 등록
사이트 로그인 후, My 페이지 또는 설정 클릭
- 등록할 사이트를 방문하여 로그인합니다. (사이트 미 회원일 경우, 가입 필요)
- 사이트 로그인 후, My 페이지 또는 설정을 누릅니다.
라이프인 오토패스워드 설정 선택
본인확인코드 확인
- "라이프인 오토패스워드 설정"을 선택한 후 "본인확인코드"를 확인합니다.
STEP3 라이프인 오토패스워드 등록 완료👏
간편로그인 사이트 추가 선택
본인확인코드 입력
사이트 등록 완료
- "라이프인" 앱의 "간편로그인 사이트 추가"에서 "본인확인코드"를 입력하고 [확인]을 누릅니다.

#이용방법

STEP1 사이트에서 라이프인 오토패스워드 로그인 선택
- "라이프인 오토패스워드"를 등록한 사이트 방문
- 로그인 시 [라이프인 오토패스워드 로그인] 선택
STEP2 아이디 입력
로그인 선택
비밀번호 생성
ID만 입력하고 "로그인"을 누르면 자동으로 비밀번호가 생성됩니다.
STEP3 "라이프인" 앱에서 확인! 로그인 완료!!
- "라이프인" 앱에서 생성된 일회용 비밀번호와 비교 후 확인을 누르세요.
6자리 숫자가 일치하는지 확인!
비밀번호를 외우고 입력할 필요 없이 웹사이트에 로그인 됩니다. 👏👏👏
